# toDoList
A simple web project using ejs

This toDoList project can be used as a virtual notepad where you can add list of works that you want to do on the day and check them out once done with the work.
JavaScript methods have been used to fetch the date from the server, hence the date is updated everyday.

Node js is used as the backend for this project.

If you are willing to run this project in a remote repository then use vscode or atom or any other compilers.
